청구항 작성도구 seoyeye 번호·인용 자동 관리

{N} 자리에 인용 항 번호가 자동 삽입됩니다. 예: 제{N} 항에 있어서, → 제3 항에 있어서,
  • [인용항 바꾸기] ⠿ 핸들을 끌어 다른 청구항 카드 위에 놓으면, 그 항을 인용하는 종속항이 됩니다. "제N 항에 있어서," 문구는 자동으로 변경됩니다.
  • [순서 바꾸기] ⠿ 핸들을 끌어 항과 항 사이 파란 띠에 놓거나, 카드 아래 ↑↓ 버튼을 누르세요. 청구항 번호는 자동으로 정렬됩니다.
  • [청구항 추가하기] "＋ 위에 항 삽입"을 누르면 새 항이 추가되고, 청구항 번호가 정렬됩니다.
  • [종속항의 종속항] "＋ 이 항의 종속항" 버튼을 누르면 종속항이 추가됩니다.
  • [다중인용] 종속항 카드의 "＋인용" 버튼으로 인용 항을 추가하면 "제1 항 또는 제3 항에 있어서,"로 합쳐지고, 연속된 3개 이상이면 "제1 항 내지 제3 항 중 어느 한 항에 있어서," 형태로 자동 변환됩니다.
  • [내보내기] 최종본은 ".docx 다운로드"를 이용하면 서식이 유지됩니다.
  • [워드 불러오기] 워드에서 수정한 .docx를 다시 가져올 때 사용하는 보조 기능입니다. 평소 작업을 이어서 할 때는 "임시저장 불러오기"를 사용하세요.
  • [작업 보관] 작업을 중간에 멈출 때는 "임시저장"으로 파일을 받아두고, 다음에 "임시저장 불러오기"로 이어서 작업하세요.
